create_substmat

Description

Create a substitution matrix based on the frequencies of amino acids and their transitions in a given .FASTA file. Assume all sequences are aligned and have the same length (with gaps).

Tests

    >>> create_substmat("baba.fasta")
    {'A': {'A': 1, 'B': -2, 'C': -2}, 'B': {'A': -2, 'B': 3, 'C': 1}, 'C': {'A': -2, 'B': 1, 'C': 3}}
    >>> create_substmat("baba.fasta")
    'The alignment contains amino acids that are not in the provided alphabet'
