entropy

Description

Shannon entropy is a measure of variation in a sequence alignment. Write a function that computes the normalized entropy for each column in a sequence alignment in a .fasta file. Normalized entropy values range between 0 and 1 (0: only one amino acid is present, 1: the 20 amino acids are equally present). Results should be presented as a list of numbers with up to two decimals.
