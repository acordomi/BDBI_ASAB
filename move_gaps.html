move_gaps

Description

Write a function move_gaps(seq1, seq2) that displays all possible alignments of the short sequence (which could be seq1 or seq2) to the long sequence (seq1 or seq2) by placing gaps in the short sequence so that it reaches the length of the long. If the alignment requires more than one gap, all gaps will go together (see tests). The function should return a list where the first element is the long sequence and the other are the different possible alignments.

Hint

  THEFASTCAT
  --AFASTCAT
  A--FASTCAT
  AF--ASTCAT
  AFA--STCAT
  AFAS--TCAT
  AFAST--CAT
  AFASTC--AT
  AFASTCA--T
  AFASTCAT--

Tests

    >>> move_gaps("THEFASTCAT", "THEFATCAT")
    ['THEFASTCAT', '-THEFATCAT', 'T-HEFATCAT', 'TH-EFATCAT', 'THE-FATCAT', 'THEF-ATCAT', 'THEFA-TCAT', 'THEFAT-CAT', 'THEFATC-AT', 'THEFATCA-T', 'THEFATCAT-']
    >>> move_gaps("THEFASTCAT", "AFASTCAT")
    ['THEFASTCAT', '--AFASTCAT', 'A--FASTCAT', 'AF--ASTCAT', 'AFA--STCAT', 'AFAS--TCAT', 'AFAST--CAT', 'AFASTC--AT', 'AFASTCA--T', 'AFASTCAT--']
    >>> move_gaps("THEFATCAT", "THEFASTCAT")
    ['THEFASTCAT', '-THEFATCAT', 'T-HEFATCAT', 'TH-EFATCAT', 'THE-FATCAT', 'THEF-ATCAT', 'THEFA-TCAT', 'THEFAT-CAT', 'THEFATC-AT', 'THEFATCA-T', 'THEFATCAT-']
    >>> move_gaps("AFASTCAT", "THEFASTCAT")
    ['THEFASTCAT', '--AFASTCAT', 'A--FASTCAT', 'AF--ASTCAT', 'AFA--STCAT', 'AFAS--TCAT', 'AFAST--CAT', 'AFASTC--AT', 'AFASTCA--T', 'AFASTCAT--']
