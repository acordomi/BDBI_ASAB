score_seqs_gap

Description

Modify the function score_seqs() in the previous exercise so that it also takes a gap penalty that computes the alignment score between two sequences (seq1 and seq1) using a match, a mismatch and a gap penalty score. Both sequeces should have the same lenght; otherwise the score should be 0.

Tests

    >>> score_seqs("THEFASTCAT", "THEFATCAT-", 1, -1, -2)
    -1
    >>> score_seqs("THEFASTCAT", "THEFATCA-T", 1, -1, -2)
    1
    >>> score_seqs("THEFA-TCAT", "THEFASTCAT", 1, -1, -2)
    7
    >>> score_seqs("THEFASTCAT", "THE", 1, -1, -2)
    0
    >>> score_seqs("THE-FASTCAT", "THE-FASTCAT", 1, -1, -2)
    8
