have_residue

Description

Write a function have_residue(fasta_file, residue, mincount) that reads a fasta file and tells how many of the sequence contain at least the number of residues specified.

Hint

Tests

    >>> have_residue("fastcats.fasta", "E", 2)
    1
    >>> have_residue("fastcats.fasta", "T", 2)
    4
    >>> have_residue("fastcats.fasta", "W", 0)
    4
    >>> have_residue("fastcats.fasta", "W", 1)
    0
    >>> have_residue("fastcats.fasta", "", 1)
    0
