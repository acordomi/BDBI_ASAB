n_nucleotide

Description

Before addressing this exercise, you should have solved dinuclotides and tetranucleotides. Write a function npeptides(res, n) that takes as input the residues to combine (res) and the length of the peptides to generate(n) and returns dipeptides, tripeptides, ... as requested. The function should call another function build_peptides(res, peptides) that enlarges the peptides provided with one letter from res. See the Tests. Do not use itertools.

Tests

    >>> build_peptides('A', [])
    ['A']
    >>> build_peptides('LY', ['A', 'L'])
    ['AL', 'AY', 'LL', 'LY']
    >>> build_peptides('', ['A'])
    ['A']
    >>> build_peptides('ALY', ['S'])
    ['SA', 'SL', 'SY']

    >>> npeptides('AY', 5)
    ['AAAAA', 'AAAAY', 'AAAYA', 'AAAYY', 'AAYAA', 'AAYAY', 'AAYYA', 'AAYYY', 'AYAAA', 'AYAAY', 'AYAYA', 'AYAYY', 'AYYAA', 'AYYAY', 'AYYYA', 'AYYYY', 'YAAAA', 'YAAAY', 'YAAYA', 'YAAYY', 'YAYAA', 'YAYAY', 'YAYYA', 'YAYYY', 'YYAAA', 'YYAAY', 'YYAYA', 'YYAYY', 'YYYAA', 'YYYAY', 'YYYYA', 'YYYYY']
    >>> npeptides('A', 10)
    ['AAAAAAAAAA']
    >>> npeptides('AYS', 2)
    ['AA', 'AY', 'AS', 'YA', 'YY', 'YS', 'SA', 'SY', 'SS']
    >>> npeptides('L', 2)
    ['LL']
