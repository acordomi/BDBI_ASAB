read_substmat_half

Description

Modify the read_substmat() function so that it reads files in the half matrix format. The function does not have to be compatible with the full format of the previous exercise.

Tests

    >>> smat = read_substmat('blosum62_half.mat')
    >>> smat['W']['W']
    11
    >>> len(smat)
    20
    >>> len(smat['L'])
    11
    >>> smat['W']['A']
    -3
    >>> 'W' in smat['A']
    False
