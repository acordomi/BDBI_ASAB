join_exons

Description

The file brca1_cds.dna contains the coding sequence of BRCA1 (each line corresponds to a coding exon). Reconstruct the complete CDS in a single string in RNA alphabet and uppercase letters. For this you will need to create a join_exons() function that returns the merged RNA string. When you are done, use the function to get the CDS from the file brca1_cds.dna and save the resulting RNA in a new file in chunks of 80 characters per line.

Tests

    >>> join_exons("brca1_cds.dna")[2000:2050]
    'ACUCAUGGAAGGUAAAGAACCUGCAACUGGAGCCAAGAAGAGUAACAAGC'
    >>> join_exons("brca1_cds.dna")[-2050:-2000]
    'CCAGAAAGGAGAGCUUAGCAGGAGUCCUAGCCCUUUCACCCAUACACAUU'
