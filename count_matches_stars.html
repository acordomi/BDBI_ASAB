count_matches_stars

Description

Write a function column_matches_stars(fasta_file) that returns a line with "*"" in columns that are fully conserved in a sequence alignment.

Hint

Use Biopython's SeqIO module

Example

    THE----FASTCAT
    THE----FA-TRAT
    THEVERYFASTCAT
    ---LASTFA-TCAT
           ** * **

Tests

    >>> count_matches_stars("fastcats.fasta")
    '       ** * **'
