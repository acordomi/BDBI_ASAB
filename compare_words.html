compare_words

Description

Write a function that reads two FASTA files, one with a single sequence and another with many sequences (it will act as the database). Score the sequence in the first file agains all sequences in the second. The score should be the sum of the individual scores of all word of length k with a threshold t using a BLOSUM62 substitution matrix. You may call your function fasta2words. For the testing you require SGMR2.fasta and protein_database.fasta.

Hint

You can open the FASTA files using Biopython 

from Bio import SeqIO
SeqIO.parse(filename, "fasta")

Tests

    >>> compare_words("SGMR2.fasta", "protein_database.fasta", t=8, k=3)[:3]
    [(6568.0, 'A0A2K5EYW3_AOTNA'), (6448.0, 'K7D4N0_PANTR'), (6447.0, 'A0A2I2ZIR7_GORGO')]
