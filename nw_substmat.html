nw_substmat

Description

Change the implementation of Needleman and Wunsch you did in nw3 so that it uses the BLOSUM62 substitution matrix instead of match/mismatch scores. The function should return the optimal score and the two aligned sequences.

Hint

You can get any substitution matrix from Biopython 
from Bio.Align import substitution_matrices
subst_mat = substitution_matrices.load("BLOSUM62")

Tests

    nw('FAT', 'FAST', -1)
    (14.0, 'FA-T', 'FAST')
    >>> nw('THEFASTCAT', 'THERAT', -4)
    (10.0, 'THEFASTCAT', 'THE---R-AT')
    >>> nw('THERAT', 'THEFASTCAT', -4)
    (10.0, 'THE---R-AT', 'THEFASTCAT')
    >>> nw('T', 'FAST', -1)
    (2.0, '---T', 'FAST')
