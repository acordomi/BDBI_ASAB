dot_matrix_window

Description

Modify dot_matrix(seq1, seq2) function to implement a window size (add a third parameter). Dot should appear ONLY at the first character of a match. When implementing the window, assume there are no inversions (diagonals always go downhill). When no window size is specified the result should be as if there was no window.

Example

For a window size of 2 the dotplot of FASTCAT and FASTRAT should be:

     FASTCAT
    Fo      
    A     o 
    T       
    R       
    A     o 
    T

Woth no window (window=1) the dotplot of FASTCAT and FASTRAT should be:

     FASTCAT
    Fo
    A o   o
    T   o  o
    R
    A o   o
    T   o  o
