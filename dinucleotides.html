dinucleotides

Description

Write a function that, given a collection of bases, returns a list with all possible dinucleotides that can be constructed combining the bases provided sorted alphabetically. Repetition is possible.

Tests

    >>> dinucleotides(['A', 'T'])
    ['AA', 'AT', 'TA', 'TT']
    >>> dinucleotides(['A', 'T', 'C'])
    ['AA', 'AT', 'AC', 'TA', 'TT', 'TC', 'CA', 'CT', 'CC']
    >>> dinucleotides('A')
    ['AA']
    >>> dinucleotides('AT')
    ['AA', 'AT', 'TA', 'TT']
