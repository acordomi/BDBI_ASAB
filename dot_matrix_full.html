dot_matrix_full

Description

Modify dot_matrix(seq1, seq2) function so thath the dots appear NOT ONLY at the first character of a match.but at the whole segment. When implementing the window, assume there are no inversions (diagonals always go downhill).

Example

For a window size of 2 and a threshold of 4 the dotplot of FASTCAT and FASTRAT should be:

     FASTCAT
    Fo
    A oo
    T  oo
    R   oo
    A     o
    T      o

Tests

    >>> dot_matrix("FASTCAT", "FATRAT", 4, 2)
    [['o', ' ', ' ', ' ', ' ', ' ', ' '], [' ', 'o', 'o', ' ', ' ', ' ', ' '], [' ', ' ', 'o', 'o', ' ', ' ', ' '], [' ', ' ', ' ', 'o', 'o', ' ', ' '], [' ', ' ', ' ', ' ', ' ', 'o', ' '], [' ', ' ', ' ', ' ', ' ', ' ', 'o']]
    >>> dot_matrix("FASTCAT", "FATRAT", 2, 4)
    A threshold larger or equal than the window does not make sense
