Palindromes

Description

Implement the three algorithms to detect palindromes described in class as 3 functions palin1(), palin2() and palin3() in a module. They should all take a sequence (str) and return a boolean. Be sure that the three algorithms work with sequences with even and odd number of elements.

Hint

Tests

    >>> palin1('ACCTAGGT')
    True
    >>> palin1('GCCGAGGT')
    False
    >>> palin1('ACCAGGT')
    False

    >>> palin2('ACCTAGGT')
    True
    >>> palin2('GCCGAGGT')
    False
    >>> palin2('ACCAGGT')
    False
    
    >>> palin3('ACCTAGGT')
    True
    >>> palin3('GCCGAGGT')
    False
    >>> palin3('ACCAGGT')
    False
