score_words

Description

Write a function that takes a list of words (all words with the same length) and performs all-to-all comparisons of words using a BLOSUM62 substitution matrix. The output should be as indicated in the tests: a dictionary of with the words as keys and a list of tuples as values that consist of the score and the aligned word, sorted from larger to smaller score.

Hint

You can take the substitution matrix from Biopython 

from Bio.Align import substitution_matrices
substitution_matrices.load("BLOSUM62")

Tests

    >>> score_words(['AAA', 'AAS', 'ASA', 'SSA', 'SSS'])
