codons2prot

Description

Write a script that reads a genetic code from a text file and makes the translation of RNA triplets into protein sequences. You will need two functions:
1) read_gencode(): should read the genetic code file and return a dictionary. You can find the standard genetic code here.
2) codons2prot(): should take a list of triplets and return a string with the protein sequence.

Tests

    >>> len(read_gencode("gencode.txt"))
    64
    >>> read_gencode("gencode.txt")['AAA']
    'K'
    
    >>> codons2prot(["AUU", "CAU", "AUC"])
    'IHI'
    >>> codons2prot(["ATT", "CAT", "ATC"])
    'IHI'
    >>> codons2prot(["CGA", "GUG", "UAA"])
    'RV/'
