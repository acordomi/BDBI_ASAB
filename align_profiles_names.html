align_profiles_names

Description

Modify the function in align_profiles so that it takes as input a single dictionary with all the sequences (as created by fasta2dict()). The sequences to align will be specified in two separate lists with only the names (that should all appear in the dictionary). For the testing you require cats_and_rats.

Hint

Use Biopython to get the substitution matrix. Use fasta2dict() 


from Bio.Align import substitution_matrices
from fasta2dict import fasta2dict

Tests

    >>> align_profiles(fasta2dict('cats_and_rats.fasta'), ['FAST_CAT', 'FAT_CAT'], ['LAST_RAT', 'FAT_RAT'], -4)
    (10.25, {'FAST_CAT': 'thefastcat', 'FAT_CAT': 'a--fa-tcat', 'LAST_RAT': 'a--ra-t---', 'FAT_RAT': 'thera-t--s'})
