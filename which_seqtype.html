which_seqtype

Description

Write a function that detects if the sequence provided is DNA, RNA or protein. Assume that sequences with only A, C, T and G are DNAs. The sequences passed are guaranteed not to have ambiguous amino acids or bases.

Tests

    >>> which_seqtype('ACTGTGTAA')
    'DNA'
    >>> which_seqtype('ACUGCUAG')
    'RNA'
    >>> which_seqtype('AHPLCTGA')
    'Protein'
    >>> which_seqtype('')
