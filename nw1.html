nw1

Description

Returns the pairwise global sequence alignment by implementing Needleman and Wunsch as in the material provided. Use provided match, mismatch and gep parameters.

Tests

    >>> nw('FAT', 'FAST', 2, -1, -1)
    ('FA-T', 'FAST')
    >>> nw('THEFASTCAT', 'THERAT', 1, -8, -4)
    ('THE-FASTCAT', 'THER-A-T---')
    >>> nw('THERAT', 'THEFASTCAT', 1, -8, -4)
    ('THE-RA-T---', 'THEF-ASTCAT')
