dot_matrix_basic

Description

Write a function dot_matrix(seq1, seq2) that returns dot matrix. This is a plot generated by putting seq1 horizonatlly and seq2 vertically. The matrix is a list of list with a dot (o) when there is a match betwen both sequences and space elswhere. See the examples.

Example

The format should be


     FASTCAT
    Fo
    A o   o
    S  o
    T   o  o
    C    o
    A o   o
    T   o  o

     FASTCAT
    T   o  o
    A o   o
    C    o
    T   o  o
    S  o
    A o   o
    Fo

Tests

    >>> dot_matrix("FASTCAT", "FASTCAT")
    [['o', ' ', ' ', ' ', ' ', ' ', ' '], [' ', 'o', ' ', ' ', ' ', 'o', ' '], [' ', ' ', 'o', ' ', ' ', ' ', ' '], [' ', ' ', ' ', 'o', ' ', ' ', 'o'], [' ', ' ', ' ', ' ', 'o', ' ', ' '], [' ', 'o', ' ', ' ', ' ', 'o', ' '], [' ', ' ', ' ', 'o', ' ', ' ', 'o']]
    >>> dot_matrix("FASTCAT", "TACTSAF")
    [[' ', ' ', ' ', 'o', ' ', ' ', 'o'], [' ', 'o', ' ', ' ', ' ', 'o', ' '], [' ', ' ', ' ', ' ', 'o', ' ', ' '], [' ', ' ', ' ', 'o', ' ', ' ', 'o'], [' ', ' ', 'o', ' ', ' ', ' ', ' '], [' ', 'o', ' ', ' ', ' ', 'o', ' '], ['o', ' ', ' ', ' ', ' ', ' ', ' ']]
