move_seq2_scores

Description

Reuse the functions move_seq2() and score_seqs() from previous exercises. Create a new function move_seq2_scores(seq1, seq2, match, mismatch, gap) that prints seq1 with the necessary gaps at the end and then the different alignments of seq2 with the corresponding alignment score to seq1.

Tests

    >>> move_seq2_scores('THEFASTCAT', 'AFASTCAT', 1, -1, -2)
    THEFASTCAT--------
    AFASTCAT---------- -28
    -AFASTCAT--------- -28
    --AFASTCAT-------- -14
    ---AFASTCAT------- -29
    ----AFASTCAT------ -28
    -----AFASTCAT----- -29
    ------AFASTCAT---- -30
    -------AFASTCAT--- -33
    --------AFASTCAT-- -32
    ---------AFASTCAT- -35
    ----------AFASTCAT -36
