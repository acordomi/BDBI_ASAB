count_matches_fasta

Description

Write a function column_matches_fasta(fasta_file) that computes how many positions (columns) have a match in a .fasta file containing only two sequences. Assume both sequences are aligned and have the same length (with gaps).

Hint

Use Biopython's SeqIO module

Tests

    >>> count_matches_fasta("cat_and_rat.fasta")
    8
