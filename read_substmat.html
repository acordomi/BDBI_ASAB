read_substmat

Description

Create a function read_substmat() that takes as input the name of a file with a substitution matrix and returns a dictionary of dictionaries with the substitution scores. Files provided should have the same format of this BLOSUM62 matrix, which is required in the tests.

Tests

    >>> smat = read_substmat('blosum62_full.mat')
    >>> smat['W']['W']
    11
    >>> smat['W']['A']
    -3
    >>> smat['A']['W']
    -3
    >>> len(smat)
    20
    >>> len(smat['L'])
    20
