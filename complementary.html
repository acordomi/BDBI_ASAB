Complementary

Description

Create a module with the 3 functions compl(seq), reverse(seq) and rev_compl(seq) that return the complementary sequence, the reverse and the reverse complementary respectively.

Tests

    >>> compl('AACTAAG')
    'TTGATTC'

    >>> reverse('AACTAAG')
    'GAATCAA'

    >>> rev_compl('AACTAAG')
    'CTTAGTT'
