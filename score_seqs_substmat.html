score_seqs_substmat

Description

Modify the function score_seqs() in score_seqs_gap to consider a substitution matrix instead of match/mismatch scores.

Hint

Use Biopython 


from Bio.Align import substitution_matrices
subst_mat = substitution_matrices.load("BLOSUM62")

Tests

    >>> score_seqs("THEFASTCAT", "THEFATCAT ", 1, "BLOSUM62")
    29
    >>> score_seqs("THEFASTCAT", "THEFATCA T", 1, "BLOSUM62")
    34
    >>> score_seqs("THEFA TCAT", "THEFASTCAT", 1, "BLOSUM62")
    52
    >>> score_seqs("THEFASTCAT", "THE", 1, "BLOSUM62")
    0
