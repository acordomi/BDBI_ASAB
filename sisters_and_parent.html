sisters_and_parent

Description

Now that you know how to print all leaves in a tree, write a function that takes a dictionary of nodes as provided by newick_file2nodes() function in tree_nodes.py and returns a list with tuples that join both sister leaves with its parent (including, the word right/left/parent), the node id and the name (only for leaves). See the tests. For the testing you will need tree_abcdef.dnd and hmgb.dnd.

Hint

Import everything from tree_nodes.py 

from tree_nodes import *

Tests

    >>> sisters_and_parent(newick_file2nodes("tree_abcdef.dnd"))
    [(('left', 5, 'B'), ('right', 4, 'A'), ('parent', 3)), (('left', 8, 'D'), ('right', 7, 'C'), ('parent', 6)), (('left', 11, 'F'), ('right', 10, 'E'), ('parent', 9))]
    >>> sisters_and_parent(newick_file2nodes("hmgb.dnd"))
    [(('left', 4, 'hmgl_wheat'), ('right', 3, 'hmgb_chite'), ('parent', 2))]
