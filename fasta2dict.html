fasta2dict

Description

Write a function fasta2dict() that reads a fasta file (using Biopython) an returns a dictionary with sequence ids (keys), and the actual sequences as Seq objects (values). For the testing you require cats.fasta and rats.fasta

Hint

You can open the fasta file using Biopython 

from Bio import SeqIO
SeqIO.parse(filename, "fasta")

Tests

    >>> fasta2dict('cats.fasta')
    {'FAST_CAT': Seq('thefastcat'), 'FAT_CAT': Seq('a--fa-tcat')}
    >>> fasta2dict('rats.fasta')
    {'A_RAT': Seq('a--rat-'), 'THE_RATS': Seq('therats')}
