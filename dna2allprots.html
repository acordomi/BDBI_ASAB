dna2allprots

Description

Write a function that returns the list all possible proteins that could be encoded from a DNA segment. Consider the 3 reading frames both on the forward and the reverse complementary sequence.

Hint

You may use Seq.reverse_complement() method from Biopython. Also in Biopython, the following dictionary contains all conversions from codons to amino acids: 

from Bio.Data import CodonTable
codon2aa = CodonTable.unambiguous_dna_by_name["Standard"].forward_table


from Bio import SeqIO
SeqIO.parse(filename, "fasta")
