residue_counts

Description

Compute the edit distance between two sequences using dynamic programming. The edit distance between two strings is the sum of insertions, deletions, or replacements required to convert one string into the other.

Tests

    >>> edit_distance('ATC', 'ATC')
    0
    >>> edit_distance('ATC', 'ATT')
    1
    >>> edit_distance('ATC', 'ATTT')
    2
    >>> edit_distance('ATC', 'CTAT')
    3
    >>> edit_distance('GCGTATGCGGCTAACGC', 'GCTATGCGGCTATACGC')
    2
