seq_profile

Description

Create a function that computes a sequence profile (position-specific scoring matrix) from a list of seqs. The function only needs to work with DNA sequences with bases (A, C, G, T). The profile is just a list with the number of times each base appears in each position of the seqs. See tests.

Tests

    >>> seq_profile([])

    >>> seq_profile(["AC", "A"])

    >>> seq_profile(["A", "A"])
    {'A': [2], 'C': [0], 'G': [0], 'T': [0]}
    >>> seq_profile(["AC", "AC"])
    {'A': [2, 0], 'C': [0, 2], 'G': [0, 0], 'T': [0, 0]}
    >>> seq_profile(["ACG", "ACC", "TCG", "TTG"])
    {'A': [2, 0, 0], 'C': [0, 3, 1], 'G': [0, 0, 3], 'T': [2, 1, 0]}
    >>> seq_profile(["ACGT", "ATCG", "TCGA", "TTGT"])
    {'A': [2, 0, 0, 1], 'C': [0, 2, 1, 0], 'G': [0, 0, 3, 1], 'T': [2, 2, 0, 2]}
