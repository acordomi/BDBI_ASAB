fasta2words

Description

Write a function that takes a FASTA file and a word length (k) and generates a dictionary with the sequence ids as the keys and an alphabetically sorted list of words of length k as values. For the testing you require thefastcat.fasta and protein_database.fasta

Hint

You can open FASTA files using Biopython 

from Bio import SeqIO
SeqIO.parse(filename, "fasta")

Tests

    >>> fasta2words("thefastcat.fasta", 3)
    {'fast_cat': ['AST', 'CAT', 'EFA', 'FAS', 'HEF', 'STC', 'TCA', 'THE']}
    >>> len(fasta2words("protein_database.fasta", 3))
    10
    >>> prots = fasta2words("protein_database.fasta", 3)
    >>> len(prots['K7D4N0_PANTR'])
    178
