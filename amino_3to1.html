amino_3to_1

Description

Write a function that reads a protein sequence in thee-letter-code format and returns the same sequence in one-letter-code format. The sequence passed can be in upper or lower case, but and always separated by '-'. Here you can find the conversion table). Write an additional function read_aminos(xxx) that read the file with the conversions and returns a dictionary.
