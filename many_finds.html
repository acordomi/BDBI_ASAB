many_finds

Description

Write a function many_finds(p, t) that returns a list with the offsets of all ocurrences of pattern p in the string t. The function should call the .find() methods multiple times.

Tests

    >>> many_finds('FAST', 'THEFASTCATTHEFASTRATAFASTRAT')
    [3, 13, 21]
    >>> many_finds('FAST', 'FAST')
    [0]
    >>> many_finds('FASTA', 'FAST')
    []
