node2leaves

Description

Write a function that takes as input a node id (i), a dictionary of nodes as provided by by newick_file2nodes() function in tree_nodes.py, and a list. The function should call itself in a recursive way. It should return a list with the names of all leaves that can be reached from such node. The initial list of leaves should be emtpy (see Tests). The idea is to start exploring left then rigth. For the testing you will need tree_abcdef.dnd and hmgb.dnd.

Hint

Import everything from tree_nodes.py 

from tree_nodes import *
