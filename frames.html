frames

Description

Define a functoin reading_frames() which takes as arguments a position (0, 1 or 2) and a DNA/RNA (str) and returns a list with the triplets that would be translated into protein. Ex: GTGATTAA could be grouped in
0: GTG ATT (AA)
1: (G) TGA TTA (A) or
2: (GT) GAT TAA

Tests

    >>> reading_frames(0, 'GTGATTAA')
    ['GTG', 'ATT']
    >>> reading_frames(1, 'GTGATTAA')
    ['TGA', 'TTA']
    >>> reading_frames(2, 'GTGATTAA')
    ['GAT', 'TAA']
