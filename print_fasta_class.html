print_fasta_class

Description

Implement a class Seq that when printed, displays the sequence id and the sequence with 60 characters per line.

Tests

    >>> record = Seq()
    >>> record.id = "FAST_CAT"
    >>> record.seq = "FASTCAT"
    >>> print(record)
    >FAST_CAT
    FASTCAT
    >>> record = Seq()
    >>> record.id = "MANY_CATS"
    >>> record.seq = "CAT"*100
