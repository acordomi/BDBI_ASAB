gc_content_cds

Description

The file brca1_cds.dna contains the coding sequence of BRCA1 (each line corresponds to a coding exon). Write a function that numerates the coding exons (starting at 1), computes the GC content of each, and returns a list of tuples (exon numbers, GC content), sorted by GC from largest to smallest GC content (see tests).

Tests

  >>> gc_conent_cds("brca1_cds.dna")[:5]
  [(22, 57.14), (21, 54.84), (19, 51.79), (10, 48.89), (20, 48.0)]
  >>> gc_conent_cds("brca1_cds.dna")[-5:]
  [(1, 37.04), (5, 36.88), (3, 36.71), (7, 36.17), (15, 34.83)]
