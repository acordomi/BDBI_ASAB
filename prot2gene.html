prot_to_gene

Description

Write a function prot2gene(uniprot_name) that, given a Uniprot Entry Name (ex. CNR1_HUMAN) or an Accession (ex. P21554), returns its (standard) gene name.

Hint

Use requests. Protein info in Uniprot can be obtained from https://rest.uniprot.org/uniprotkb/XXXXXX.txt, with XXXXXX being the Uniprot Entry Name or the Accession.

Tests

    >>> prot2gene("PACR_BOVIN")
    'ADCYAP1R1'
    >>> prot2gene("")

    >>> prot2gene("AAAAA_HUMAN")

    >>> prot2gene("P21554")
    'CNR1'
    >>> prot2gene("CNR1_HUMAN")
    'CNR1'
