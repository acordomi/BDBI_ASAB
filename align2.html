align2

Description

Do a pairwise alignment of two sequences from a .FASTA file containing multiple homologous sequences. The sequences to align are indicated by Uniprot name, which appears as part of the sequence id.

Tests

    >>> align2("ppar.fasta", "PPARD_XENLA", "PPARG_CANLF", -4)[0]
    876.0
    >>> align2("ppar.fasta", "AAAA_AAAA", "PPARG_CANLF", -4)[0]
    0
