gc_content

Description

Write a function that returns the GC content of a sequence. The content should be in percentage (without the % symbol) and rounded up to two decimals.

Tests

    >>> gc_content("GCGAGCTAGCTAGCGATCAGCATCATCACATGCATCGTGTA")
    51.22
    >>> gc_content("CGCGCGCGCCCGCGCGCGCGCGCCCCGGGCGCGCGCGCGCGCG")
    100.0
    >>> gc_content("")
    0.0
