count_matches_multiple

Description

Write a function column_matches_multiple(fasta_file) that computes how many positions (columns) have a match in all sequences in a .fasta file with any number of sequences. Assume all sequences are aligned and have the same length (with gaps).

Hint

Use Biopython's SeqIO module

Tests

    >>> count_matches_multiple("fastcats.fasta")
    5
