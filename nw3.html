Description

Change the implementation of Needleman and Wunsch you did in nw2.

Hint

In nw1 you had ... 

    >>> nw('THEFASTCAT', 'THERAT', 1, -8, -4)
    ('THE-FASTCAT', 'THER-A-T---')
    >>> nw('THERAT', 'THEFASTCAT', 1, -8, -4)
    ('THE-RA-T---', 'THEF-ASTCAT')

In nw2 you had ... 

    >>> nw('THEFASTCAT', 'THERAT', 1, -8, -4)
    ('THEF-ASTCAT', 'THE-RA-T---')
    >>> nw('THERAT', 'THEFASTCAT', 1, -8, -4)
    ('THER-A-T---', 'THE-FASTCAT')

which are both different from the tests you have below

Tests

    >>> nw('THEFASTCAT', 'THERAT', 1, -8, -4)
    ('THEFASTCAT', 'THE----RAT')
    >>> nw('THERAT', 'THEFASTCAT', 1, -8, -4)
    ('THE----RAT', 'THEFASTCAT')
