read_substmat_sym

Description

Modify the read_substmat() function again so that it still reads files in the half matrix format, but it returns a dictionary of dictionaries that allows to look up for the substitiution scores in both directions (i.e. [A][B] and [B][A]).

Tests

    >>> smat = read_substmat('blosum62_half.mat')
    >>> smat['W']['W']
    11
    >>> len(smat)
    20
    >>> len(smat['L'])
    20
    >>> smat['W']['A']
    -3
    >>> 'W' in smat['A']
    True
