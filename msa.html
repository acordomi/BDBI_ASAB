Description

And finaly, the MSA! We just need to merge all the pieces to get a progressive multiple sequence alignment. Start by copying your function node2splits. You will need to do a few modifications: 1) name it node2splits_align(), 2) add two additional objects: seqs (a dictionary from fasta2dict) and gep, 3) be sure to call align_profiles. For the testing you require hmgb.dnd and hmgb.fasta.

Hint

Use tree_nodes and some of your previous functions. 

from tree_nodes import *
from fasta2dict import fasta2dict
from align_profiles_names import align_profiles
