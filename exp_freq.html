exp_freq

Description

Write a function exp_freq(biopython_gencode) that computes the expected frequency for each amino acid considering all possible triplets in a speficic genetic code. Genetic codes are implemented in Biopython (Standard by default).

Hint

import CodonTable from Bio.Data

The dictionary for the "Standard" genetic code is in CodonTable.unambiguous_dna_by_name['Standard'].back_table

The list of all the genetic codes implemented in Biopython can be seen in CodonTable.unambiguous_dna_by_name.keys()

Tests

    >>> len(exp_freq("Standard"))
    20
    >>> freqs = exp_freq("Standard")
    >>> freqs['I']
    0.046875
    >>> freqs = exp_freq('Yeast Mitochondrial')
    >>> freqs['I']
    0.03125
    >>> freqs = exp_freq("Plant Plastid")
    >>> freqs['I']
    0.046875
    >>> freqs = exp_freq()
    >>> freqs['I']
    0.046875
