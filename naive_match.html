naive_match

Description

Write a function naive_match(p, t) that returns a list with the offsets of all ocurrences of pattern p in the string t. Implement the Naive Match Algorithm as described in the slides. Do not use .find(). Slide the pattern over text and compare letters one by one searching for a match.

Tests

    >>> naive_match('FAST', 'THEFASTCATTHEFASTRATAFASTRAT')
    [3, 13, 21]
    >>> naive_match('FAST', 'FAST')
    [0]
    >>> naive_match('FASTA', 'FAST')
    []
