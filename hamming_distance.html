Hamming distance

Description

Given any two sequences of equal length, the Hamming distance is the number of positions at which corresponding symbols are different. Write a function that returns the Hamming distance between two sequences.

Tests

    >>> hamming_distance('AGCTATC', 'AGCTCTC')
    1
    >>> hamming_distance('AGCTATC', 'AGCTCC')

    >>> hamming_distance('AGCTATC', 'AAAAAAA')
    5
