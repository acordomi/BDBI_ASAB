overlaps

Description

Write a fuction count_ov(sequence, motif) that works similar to str.count() method but is aware of overlaps between motifs (see Tests). count_ov should return a tuple with both the counts and a list with the positions where motifs start.

Tests

    >>> 'GATATAT'.count('ATAT')
    1
    >>> count_ov('GATATAT', 'ATAT')
    (2, [1, 3])
    >>> count_ov('GATATATATATGCATATATACTT', 'ATAT')
    (6, [1, 3, 5, 7, 13, 15])
    >>> count_ov('GAT', 'ATAT')
    (0, [])
    >>> count_ov('GAT', '')
    (0, [])
