leaves

Description

Figure out how to to print all leaves in a tree. A leaf is a node with no descendants. Write a function that takes a dictionary of nodes as provided by newick_file2nodes() function in tree_nodes.py and returns a list of tuples containing the node id and the name for all leaves in the tree. For the testing you will need tree_abcdef.dnd and hmgb.dnd .

Hint

Import everything from tree_nodes.py 

from tree_nodes import *

Tests

    >>> leaves(newick_file2nodes("tree_abcdef.dnd"))
    [(4, 'A'), (5, 'B'), (7, 'C'), (8, 'D'), (10, 'E'), (11, 'F')]
    >>> leaves(newick_file2nodes("hmgb.dnd"))
    [(3, 'hmgb_chite'), (4, 'hmgl_wheat'), (5, 'hmgl_trybr'), (7, 'hmgt_mouse')]
