move_gaps_scores

Description

Reuse the functions move_gaps() and score_seqs() from previous exercises. Create a new function move_gaps_scores(seq1, seq2, match, mismatch, gap) that prints the long sequence (seq1 or seq2) and then the short sequence (seq1 or seq2) with the corresponding alignment score to the long sequence.

Tests

    >>> move_gaps_scores('THEFASTCAT', 'AFASTCAT', 1, -1, -2)
    THEFASTCAT
    --AFASTCAT 2
    A--FASTCAT 2
    AF--ASTCAT 0
    AFA--STCAT -2
    AFAS--TCAT -4
    AFAST--CAT -6
    AFASTC--AT -8
    AFASTCA--T -10
    AFASTCAT-- -12
    >>> move_gaps_scores('AFASTCAT', 'THEFASTCAT', 1, -1, -2)
