count_matches

Description

Given two sequences seq1 and seq2, count the number of positions in which seq1 and seq2 contain the same residue (as if it was a sequence alignment). Display both sequences with an additional line containing * when there is a match. Assume seq1 and seq2 have the same length.

Hint

Examples

Ex1:
With sequences defined as the following variables:
seq1 = "AAAAAAAA"
seq2 = "AGCTATAA"

The output should be:
AAAAAAAA
AGCTATAA
*   * ** 
found 4 matches as indicated by *

Ex2: 
With sequences defined as the following variables:
seq1 = "AAAAAAAA"
seq2 = "TTTTTTTT"

The output should be:
AAAAAAAA
TTTTTTTT

found 0 matches as indicated by *
