column_counts

Description

Write a function that takes a fasta file and a column position and returns a dictionary with the counts of each of the residues (or gaps) in that specific column in the alignment. Columns start at 0.

Tests

    >>> column_counts("fastcats.fasta", 0)
    {'T': 3, '-': 1}
    >>> column_counts("fastcats.fasta", 3)
    {'-': 2, 'V': 1, 'L': 1}
    >>> column_counts("fastcats.fasta", 7)
    {'F': 4}
