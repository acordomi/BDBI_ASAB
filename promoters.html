promoters

Description

In 1983, Hawley & McClure identified that middle gene promoters in bacteriophage T4 had two consensus sequences: a MotA box with the sequence TTGATT centered at position -30 and the sequence TAnAnT (n: any base) centered at position -10 relative to the transcriptional start site. The objective here is to create a script that reads the genome sequence and counts the number of matches. We need to consider matches in the forward strand and in the complementary strand. We thus need to obtain the reverse complementary sequence (re-use the function rev_compl() from complementary. You may need to recall the IUPAC base alphabet.

You will need to create two functions:

read_fasta(filename) to read the genome in fasta format and return a the sequence as a string.

find_matches(motif, dna) to find the motif in the sequence string and return two lists with the motifs found in the forward and in the reverse strand

Hint

A copy of the fasta file can be taken from here. You may use regular expressions (re module). In total (both strands), you should identify 222 TTGATT and 2845 TAnAnT motifs.
