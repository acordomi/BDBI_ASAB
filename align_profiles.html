align_profiles

Description

Makes a profile alignment of two groups of aligned sequences. Sequences should be provided as dictionaries, as created by fasta2dict(). Use a BLOSUM 62 and provide the gep as a parameter. For the testing you require cats.fasta and rats.fasta.

Hint

Use Biopython to get the substitution matrix. Use fasta2dict() 


from Bio.Align import substitution_matrices
from fasta2dict import fasta2dict

Tests

    >>> align_profiles(fasta2dict('cats.fasta'), fasta2dict('rats.fasta'), -4)
    (10.25, {'FAST_CAT': 'thefastcat', 'FAT_CAT': 'a--fa-tcat', 'A_RAT': 'a--ra-t---', 'THE_RATS': 'thera-t--s'})
