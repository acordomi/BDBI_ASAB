ungap_fasta

Description

Remove columns with a high percentage of gaps from a .FASTA file. and save the modified sequences to a new file. Assume all sequences are aligned and have the same length (with gaps).

Args: input_file (str): The path to the input FASTA file. output_file (str): The path to the output FASTA file. min_perc (float, optional): The minimum percentage of gaps allowed in a column. Columns with a higher percentage of gaps will be removed. Defaults to 10.

Returns: int: The length of the modified sequences after removing columns with many gaps.

Tests

    >>> ungap_fasta("gpcrs.fasta", "gpcrs_ungapped.fasta", 10)
    277
    >>> ungap_fasta("gpcrs.fasta", "gpcrs_ungapped.fasta")
    277
    >>> ungap_fasta("gpcrs.fasta", "gpcrs_ungapped.fasta", 5)
    262
