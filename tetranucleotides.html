tetranucleotides

Description

Write a function that, given a collection of bases, returns a list with all possible tetranucleotides that can be constructed combining the bases provided sorted alphabetically. Repetition is possible.

Tests

    >>> tetranucleotides(['A', 'C'])
    ['AAAA', 'AAAC', 'AACA', 'AACC', 'ACAA', 'ACAC', 'ACCA', 'ACCC', 'CAAA', 'CAAC', 'CACA', 'CACC', 'CCAA', 'CCAC', 'CCCA', 'CCCC']
    >>> tetranucleotides('CA')
    ['AAAA', 'AAAC', 'AACA', 'AACC', 'ACAA', 'ACAC', 'ACCA', 'ACCC', 'CAAA', 'CAAC', 'CACA', 'CACC', 'CCAA', 'CCAC', 'CCCA', 'CCCC']
    >>> tetranucleotides('A')
    ['AAAA']
