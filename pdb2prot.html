pdb2prot

Description

Write a function pdb2prot() that connects to your mysql database in protein PDBs and returns a dictionary with the PDB ids as keys and the protein entry names as values.

Hint

You may use sqlalchemy and pandas.

Tests

    >>> pdbs = pdb2prot()
    >>> pdbs['5C1M']
    'OPRM_MOUSE'
    >>> len(pdbs)
    164
    ''
